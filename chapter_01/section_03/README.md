# Question #1

### What is a data?

Data is any information that can be moved, processed, or stroed by a computer.

# Question #2

### What is a value?

A value is a letter (e.g. `a`), number (e.g. `5`), text (e.g. `Hello`), or instance of some other useful concept that can be represented as data.

# Question #3

### What is an object?

An object represents a region of storage (typically RAM or a CPU register) that can hold a value.

# Question #4

### What is a variable?

A variable is an object that has a name.

# Question #5

### Waht is an identifier?

An identifier is the name that a variable is accessed by.

# Question #6

### Waht is a data type used for?

A data type determines what kind of value (e.g. a number, a letter, text, etc) the object will store.

# Question #7

### What is an integer?

An integer is a number that can be written without a fractional component.
